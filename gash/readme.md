# Username GEN (With many features) - Made For Gash
Generates usernames of your choice as well has many features like custom wordlist maker, username gen, token checker

# Configuration
/data/tokens.txt - (Input your tokens)
/functions/helpers/configure - (Change all your configuration)

## Installing packages (if i forget anything and it says module not found just do the same thing)
- py -m `pip` install aiohttp
- py -m `pip` install discord
- py -m `pip` install colorama
- py -m `pip` install random

# This is how u will install packages that i forgot to do
```py -m pip install <package>``` 

# Custom Worldist (Custom.txt > Worldist.txt)
Once you generate your wordlist just move it into data/wordlist.txt
or just use a random wordlist and input it into data/wordlist.txt

# Runnning the generator
- py runner.py
